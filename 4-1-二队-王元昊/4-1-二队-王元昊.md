```py
import json

with open('1.json', 'r') as f:
    data = json.load(f)

flag = open('flag.txt', "w")

for i in range(0, len(data['data'])-1, 4):
    x = str(i)
    y = str(i+1)
    z = str(i+2)
    m = str(i+3)
    # print('({},{},{})'.format(data['data'][x], data['data'][y], data['data'][z]))
    flag.write('{},{},{},{}\n'.format(data['data'][x], data['data'][y], data['data'][z], data['data'][m]))
```

> I'm giving you a canvas image data! > width, height = [500, 80]

题目提示这个，要这个500*80的画布，尝试吧得到的数据在画布体现

```py
from PIL import Image
x =80
y = 500
img = Image.new("RGBA",(x,y))
file = open('./flag.txt','r')

for width in range(0,x):
   for height in range(0,y):
       line = file.readline()
       rgb = line.split(',')
       img.putpixel((width,height),(int(rgb[0]),int(rgb[1]),int(rgb[2]),int(rgb[3])))
img.save('flag.png')
```
